# BUG-003 — Неверный body ответа при успешной регистрации

**ID:** DEV-16
**Статус:** К выполнению
**Severity:** Minor
**Priority:** Medium

## Окружение

| Параметр | Значение |
|----------|----------|
| Окружение | Local |
| Метод | POST |
| URL | http://127.0.0.1:8000/api/auth/register |
| Инструмент | Postman 12.7.0 |
| OS | Windows 10 Pro |

## Описание

При успешной регистрации сервер возвращает только сообщение об успехе вместо данных созданного пользователя. Согласно документации ответ должен содержать id и name пользователя.

## Шаги воспроизведения

1. Открыть Postman
2. POST на http://127.0.0.1:8000/api/auth/register
3. Заголовок: Content-Type: application/json
4. Body: { "name": "testuser123", "password": "StrongPass123" }
5. Нажать Send

## Ожидаемый результат

- Статус-код: 201 Created
- Body:
  { "user": { "id": 1, "name": "testuser123" } }

## Фактический результат

- Статус-код: 201 Created
- Body:
  { "message": "User created successfully" }
